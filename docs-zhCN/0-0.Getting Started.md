### 安装
#### 需要准备
##### Chocolatey
chocolatey是windwos平台的包管理工具，与yum和apt-get类似。
##### Python 2
可以这样安装：
```
choco install python2
```
##### Node
```
choco install nodejs.install
```
##### React Native Command Line Tools
```
npm install -g react-native-cli
```
##### Android Studio
版本>2.0
android SDK 和模拟器
